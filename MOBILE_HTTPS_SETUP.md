# 手机访问HTTPS设置指南

## 问题说明
手机浏览器要求使用HTTPS才能访问麦克风，HTTP无法使用语音功能。

## 解决步骤

### 1. 生成SSL证书
```bash
chmod +x generate-cert.sh
./generate-cert.sh
```

### 2. 启动HTTPS服务器
```bash
npm run dev:https
```

### 3. 手机访问设置

#### iOS Safari:
1. 访问 `https://192.168.31.224:3000`
2. 会看到"此连接不是私密连接"警告
3. 点击"显示详细信息"
4. 点击"访问此网站"
5. 点击"访问"确认

#### Android Chrome:
1. 访问 `https://192.168.31.224:3000`
2. 会看到"您的连接不是私密连接"警告
3. 点击"高级"
4. 点击"继续前往 192.168.31.224（不安全）"

### 4. 授权麦克风权限
首次点击麦克风按钮时，浏览器会请求麦克风权限，点击"允许"。

## 注意事项
- 确保手机和电脑在同一WiFi网络
- 如果IP地址变化，需要重新生成证书
- 证书有效期365天

## 故障排除

### 如果仍然无法访问麦克风：
1. 检查浏览器设置中的麦克风权限
2. 尝试清除浏览器缓存
3. 重启浏览器

### 如果证书错误：
```bash
rm -rf .cert
./generate-cert.sh
```
