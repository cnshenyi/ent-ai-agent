# 许庚医生 - 耳鼻喉AI助手

专业的耳鼻喉科AI智能助手应用，基于Next.js和DeepSeek AI构建。

## 功能特性

- 💬 AI智能咨询 - 与许庚医生AI进行实时对话
- 🔍 症状自查 - 快速症状评估工具
- 📋 就诊历史 - 保存和查看历史咨询记录
- 🎤 语音输入 - 支持语音转文字输入
- 📱 移动优化 - 完美适配手机端

## 快速开始

1. 安装依赖：
```bash
npm install
```

2. 配置API密钥：
在 `.env.local` 文件中设置您的DeepSeek API密钥：
```
DEEPSEEK_API_KEY=your_api_key_here
```

3. 运行开发服务器：
```bash
npm run dev
```

4. 打开浏览器访问 [http://localhost:3000](http://localhost:3000)

## 获取DeepSeek API密钥

访问 [https://platform.deepseek.com](https://platform.deepseek.com) 注册并获取API密钥。

## 技术栈

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- DeepSeek AI API
